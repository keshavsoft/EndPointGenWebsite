EndPointGen Extension

A simple Visual Studio Code extension that instantly initializes an Express.js backend project using a right-click action.

Developer Workflow

Developers only need to open an empty folder inside Visual Studio Code, right click on the folder, and select the Express scaffold option.

Empty Folder → Right Click → Click Express Scaffold → Express Backend Generated

The extension automatically creates the backend structure, initializes Express.js files, and prepares the project for development.
