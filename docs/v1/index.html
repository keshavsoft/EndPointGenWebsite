# Frequently Asked Questions

How do I access the top toolbar dropdown selections like "get" and "delete"?

Ensure you have the express api actions generate extension installed. When opening an authorized endpoint configurations file (e.g., end-points.js), the action triggers automatically hook directly into your active window's configuration canvas menu panel.

What databases does the generated service structure support?

The modular layout decouples routers and controllers cleanly. The generated services populate baseline structural signatures that map flexibly into any primary relational database or document data store configuration layer.

Can I introduce custom action hooks alongside boilerplate methods?

Yes! Simply create a corresponding sibling subdirectory matching your action layout inside your target collection workspace, complete with your standard standalone module handling logic signatures.
