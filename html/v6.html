EndPointGen

EndPointGen is a Visual Studio Code extension designed to accelerate backend API development using Node.js, Express, and KeshavSoft architecture patterns. The extension helps developers automatically generate routes, endpoints, CRUD operations, modular folder structures, and service layers directly from VS Code.

Why Developers Should Use EndPointGen

Rapid Backend Development

Generate production-ready CRUD APIs within seconds instead of manually creating repetitive route and controller files.

Clean Architecture

Automatically separates routes, services, controllers, middleware, and errors into modular components.

Developer Productivity

Removes boilerplate coding and allows developers to focus more on business logic and application features.

Consistent Structure

Every endpoint follows a predictable and scalable structure, making collaboration easier for teams.

Core Features

  • ShowAll - Retrieve all records from a table.
  • Find - Fetch a single record using a primary key.
  • Filter - Return multiple matching records based on conditions.
  • Insert - Add new records with automatic primary key generation.
  • Delete - Remove records safely using primary key validation.
  • Alter - Update existing records dynamically.
  • Automatic Route Generation - Creates Express routes automatically.
  • HTTP Request Templates - Generates rest.http files for quick API testing.

Generated Backend Architecture

EndPointGen follows a layered backend architecture that separates responsibilities into independent modules. This improves scalability, maintainability, debugging, and team collaboration.

Routes Layer

Handles API endpoint registration and request routing using Express Router.

Controller Layer

Processes requests and responses while coordinating business operations.

Service Layer

Contains database interaction and kschema query or mutate operations.

Middleware Layer

Performs request validation, body checks, and error handling before processing.

KSchema Query & Mutate Architecture

The extension integrates with @keshavsoft/kschema to simplify data management using structured query and mutate APIs.

01

Query Operations

Includes findAll(), findByPk(), filterByPk(), and filterByColumns() for retrieving records.

02

Mutate Operations

Includes insertGenPk(), deleteWithChecks(), and updateWithChecks() for modifying data.

03

Structured JSON Storage

Uses JSON-based table storage which makes development lightweight and easy to understand.
